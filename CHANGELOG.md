## [0.3.0] - 2025-02-12

### Added

- **Python code emitter**: Generate Python 3.13+ validation modules with `--target python` or `--target py`
  - Zero runtime dependencies (standard library only)
  - Validated against the official JTD compliance suite
  - Use case: Python services and scripts needing JTD validation

### Changed

- README: Added Python to features, CLI usage, workflows, and code examples
